## ‚öôÔ∏è Development & Deployment

This project uses **Vite** for the build process and is deployed using the `/dist` folder via **GitHub Pages**.

### Prerequisites

* Node.js (LTS recommended)
* npm or yarn

### Local Setup

1.  Clone the repository:
    ```bash
    git clone [https://github.com/Ramazancelikgtu/Countdown-website.git](https://github.com/Ramazancelikgtu/Countdown-website.git)
    ```
2.  Navigate to the project folder:
    ```bash
    cd Countdown-website
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```
4.  Run the project locally (for development):
    ```bash
    npm run dev
    ```

### Building for Production

To create the optimized, ready-to-deploy files:

```bash
npm run build
